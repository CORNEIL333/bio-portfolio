# Rapport Bio-Portfolio

## Description

Projet réalisé dans le cadre de l'apprentissage de Git.

## Fonctionnalités

- Création d'un dépôt Git
- Gestion des commits
- Gestion des branches
- Simulation d'Issue
- Simulation de Pull Request
- Résolution de conflit
- Fusion des branches

## Logique du projet

Le projet présente un profil étudiant simple.
Une branche spécifique a été utilisée pour ajouter les compétences.
Un conflit a été simulé puis résolu afin de démontrer la maîtrise du workflow Git.
